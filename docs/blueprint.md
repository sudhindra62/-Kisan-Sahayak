# **App Name**: KisanSahayak

## Core Features:

- Farmer Profile Input: Collects farmer's land size, location (state/district), crop type, irrigation type, and annual income through a simple form.
- Scheme Eligibility Analyzer: Analyzes farmer's profile data and matches it against the Firestore database of government schemes using a rule-based matching engine.
- Eligible Schemes Display: Displays a list of government schemes for which the farmer is eligible, showing scheme name, benefits, and a summary of eligibility criteria.
- Step-by-Step Guide: Provides detailed, step-by-step instructions on how to apply for each displayed scheme, including links to official portals and required documents.
- Schemes Database: Stores all the government schemes with all of the eligibility parameters
- Rule-Based Tool: An internal reasoning tool that checks the eligibility criteria to identify the right scheme for each farmer.
- Database storage: A Firestore database to store the application.

## Style Guidelines:

- Primary color: Deep Teal (#008080) for a professional and trustworthy feel.
- Background color: Off-white (#F8F8FF) to provide a clean and sophisticated backdrop.
- Accent color: Burnt Orange (#CC6633) for highlighting important information and calls to action with warmth and sophistication.
- Body font: 'Roboto' for a clean and easily readable experience.
- Headline font: 'Montserrat' for a modern and professional look.
- Use refined, minimalist icons to represent different aspects of farming and scheme eligibility.
- Ensure a clean, spacious layout with ample whitespace to promote readability and reduce clutter.
- Incorporate subtle, professional animations such as smooth transitions and micro-interactions to enhance user engagement.
